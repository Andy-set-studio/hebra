## Getting started

### Method one: One-Click Deploy to Netlify

You can [deploy Hebra to Netlify with one click][deploy-to-netlify] and youâ€™ll be up and running in minutes!

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)][deploy-to-netlify]

### Method two: Clone / Fork

1. Clone or fork this repo: `git clone https://github.com/hankchizljaw/herbra`
2. `cd` into the project directory and run `npm install`
3. Once all the dependencies are installed run `npm start`
4. Open your browser at `http://localhost:8080` and away you go!

## Terminal commands

### Serve the site locally

```bash
npm start
```

### Build a production version of the site

```bash
npm run production
```
